# git show

## View files of last commit

{% highlight sh %}
$ git show REVISION:path/to/file
{% endhighlight %}

Or, redirect it to a file.

{% highlight sh %}
$ git show REVISION:path/to/file > output.txt
{% endhighlight %}

# git log

## View remote commit log

{% highlight sh %}
$ git log remote_name/branch_name
{% endhighlight %}

E.g, `git log origin/master`.
